# Pasta de Configuração

Cole aqui o seu arquivo `service-account.json` gerado pelo Firebase Console.

⚠️ **IMPORTANTE**: Este arquivo já está no `.gitignore` e não deve ser commitado no repositório por questões de segurança.

**Instruções para obtenção:**
1.  Vá para o [Firebase Console](https://console.firebase.google.com/).
2.  Clique no ícone de engrenagem (Configurações do Projeto).
3.  Vá em **Contas de Serviço**.
4.  Clique em **Gerar nova chave privada**.
5.  Renomeie o arquivo baixado para `service-account.json` e coloque-o nesta pasta.
