---
description: Como iniciar o Dashboard Streamlit localmente
---

Para visualizar o painel de controle e a análise exploratória:

// turbo
1. Execute o comando no seu terminal:
```powershell
streamlit run app.py
```

2. Abra o navegador no endereço indicado (geralmente `http://localhost:8501`).

⚠️ **Importante**: Certifique-se de que o arquivo `config/service-account.json` está presente para que a conexão com o Firebase funcione localmente.
